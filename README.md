# TeleCatBot
Telegram Bot with lot of cats =(^_^)=


Requirements to launch:

- Token from http://thecatapi.com/docs.html
- Redis for like / dislike, to know wich was last image
- Telegram bot token


Python requeriments:

pip install python-telegram-bot redis


Environment vars:

TELEGRAM_TOKEN
THECATAPI_KEY
REDIS_HOST
REDIS_PORT
REDIS_DB
